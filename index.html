Fluxbot

The small, extendable IRC bot for Node.js

Follow me onGitHub

A Weekend of Code project by whiskers75.

Fluxbot is a lightweight, easy to use IRC bot. It is extensible, like a supybot, and can perform all normal bot tasks easily, with a permission system to regulate who can do what.

To get running, set up a Redis database somewhere, configure config.json to point to it, and run bot.js. While you're there, configure your IRC server, nick, command prefix and channels to join. Optionally, you may want to add a password for authentication to NickServ.

Fluxbot can be extended in the form of plugins, which can contain custom code to run on start and commands, with integrated permissions and help.

Want to see a live fluxbot? Come join us on irc.freenode.net, channel #fluxbot.
